01-Emmet

Plugin for faster HTML and CSS coding.

01-Child >

Creates child elements (nesting)

Example:

ul>li>a ↔️

Result:

<ul> <li> <a href=""></a> </li> </ul>

02-Sibling +

Creates sibling elements.

Example:

header+section+footer ↔️

Result:

<header></header> <section></section> <footer></footer>

03-Multiplication *

Repeats elements multiple times.

Example:

ul>li*3 ↔️

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04-Class .

Creates a class attribute.

Example:

div.container ↔️

Result:

<div class="container"></div>

05-Classes ..

Creates multiple class attributes.

Example:

div.color1.color2.color3 ↔️

Result:

<div class="color1 color2 color3"> </div>

06-ID #

Creates an ID attribute.

Example:

nav#menu ↔️

Result:

<nav id="menu"></nav>

07-Text { }

Inserts text content.

Example:

a{Click Here} ↔️

Result:

<a href="">Click Here</a>

08-Custom Attr [ ]

Creates custom attributes.

Example:

a[href=http://www.google.com] ↔️

Result:

<a href="http://www.google.com"> </a>

09-Numbering $

Adds consecutive numbers.

Example:

ul>li.num$*3 ↔️

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10-Invert Number @-

Inverts numbering order.

Example:

ul>li.num$@-*3 ↔️

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11-Base Numbering @n

Defines a base number for sequences.

Example:

ul>li.num$@3*4 ↔️

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climp Up ^

Moves up one level.

Example:

section>article>div^p ↔️

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping ( )

Groups elements together.

Example:

(header>nav)+(section>article) ↔️

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-Lorem lorem

Generates placeholder text.

Example:

lorem5 ↔️

Result:

Lorem ipsum dolor sit amet.

15-Documents !

Creates various HTML document types.

Example:

! ↔️

Result:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> </html>

16-Meta Tags ( )

Generates meta tags.

Example:

meta:utf ↔️

Result:

<meta http-equiv="Content-Type">
